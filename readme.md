# TO-DO LIST (PhygitalTech Task)

📋 Simple Task Management Application

📌 Objective  
This project is a responsive To-Do List web application that allows users to manage their tasks efficiently. It provides basic CRUD operations, supports task filtering, drag reordering, and data persistence using LocalStorage.


✅ Features

1️⃣ Task Creation (Create) 
- Users can add new tasks with:  
  - Title (required)

2️⃣ Task Listing (Read)
- Tasks are displayed under two sections:  
  - 📋 Pending Tasks  
  - ✅ Completed Tasks

3️⃣ Edit Task (Update) 
- Users can edit the Title of tasks.

4️⃣ Delete Task (Delete)
- Users can delete any task from either section.

5️⃣ Filter Tasks  
- Users can filter tasks by: All, Completed, Pending

6️⃣ Drag & Drop Functionality  
- Users can drag and drop tasks to reorder them.  
- Task order persists using LocalStorage.

7️⃣ Persistent Storage
- All tasks and their statuses persist using LocalStorage.  
- Data remains even after page reload.

8️⃣ Dynamic Messages  
- Displays messages when:  
  - No tasks are added → "Enter your tasks"  
  - No completed tasks → "No task completed"  
  - No pending tasks → "No tasks are pending"

🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript
- LocalStorage for data persistence
